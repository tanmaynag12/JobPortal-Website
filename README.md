# Job Portal Website (MERN Stack)

## Overview
A full-stack job portal application where users can create accounts, browse jobs, and apply to jobs by uploading resumes. Recruiters can post and manage job listings. This project simulates a real-world job application workflow with authentication, job search, and application handling.

## Tech Stack
- Frontend: React (Vite), JavaScript, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- File Storage: Cloudinary
- Deployment: Render 

## Features
- User authentication (signup/login)
- Browse and search job listings
- Apply to jobs with resume upload
- Recruiter can post and manage job listings
- REST APIs for jobs and applications
- Basic validation and error handling

## Project Structure
- /frontend – React frontend (Vite)
- /backend – Node.js + Express backend
